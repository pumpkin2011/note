### C中的数据类型  
- 基本数据类型  
  - 整形 int  
  - 字符型 char  
  - 实型  
    - 单精度实型 float  
    - 双精度实型 double  

- 构造类型  
  - 数组类型  
  - 结构类型 struct  
  - 联合类型 union  
  - 枚举类型 enum  

- 指针类型  

- 空类型(无值类型) void  

### C++中新增的  
- bool    
|  -  |  逻辑类型 |  真  |  假  |
|:----|:------ |:------:| ----:|
| C   |   没有   | 非0  |  0 |
| C++ | bool    | true | false |

### 初始化 
- C  
`int x = 1024;` 复制初始化  

- C++  
`int x = 1024;` 复制初始化  
`int x(1024);` 直接初始化
