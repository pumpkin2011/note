# 常用的调试工具
- wireshark
- tcpdump
- strace/dtruss: 系统调用跟踪工具
- lsof: 列出文件打开情况
- valgrind: 查内存泄露
- ltrace: 查询库调用
- [NCurses Disk Usage](https://dev.yorhel.nl/ncdu): 查看文件占用
- ngrok: 反射代理，外网访问局域网
